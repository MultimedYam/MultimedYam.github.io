Lords of Minecraft Voice Communication Directory

Servers

This directory lists all available voice servers that are provided by individuals of the districts. These are independetly run voice servers and have no affiliation to BufalloWizards.com and Lords of Minecraft

Connecting to a server has deliberatly made easy. If you wish to connect to any district which offers a voice server, you will need to connect using the IP address

[district].lomvoice.com
